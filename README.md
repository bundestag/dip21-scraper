## Getting started

Clone the git repo & run the project
```
git clone git@github.com:bundestag/dip21-scraper.git
cd dip21-scraper
yarn install
```

### Compile and start
```
yarn start
```

- Select Periods
- Select OperationTypes

### Sart with Parameters
```
yarn start --periods 18,19 --operationtypes 100,200
```

### Test Project
```
yarn lint
```
